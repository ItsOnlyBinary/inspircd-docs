### `<pid>`

The `<pid>` tag defines the location of the pidfile. This tag can only be defined once.

Name | Type | Default Value     | Description
---- | ---- | ----------------- | -----------
file | Text | data/inspircd.pid | The location of the pidfile.

#### Example Usage

```xml
<pid file="data/inspircd.pid">
```
